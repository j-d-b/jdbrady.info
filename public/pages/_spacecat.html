SPACECAT is a precision platform game, which involves navigating our hero, the spacecat, to the exit portal.

It requires measured, precise movement and visual awareness, as well as speed.

SPACECAT was created with imcinerney for a functional programming course in 2016. I later put more work into cleaning it up and adding features after the course for a v1.1 release in early January, 2017.

Instructions are included in the zip.
