YT WorldTrends is a single-page web app facilitating the exploration of trending video content across the globe.

Built for my senior seminar in college, in 2017, YT WorldTrends site was my first time using JavaScript and first web project in general.

Select a country to explore top trending videos sourced from youtube for that region. Content is dynamically updated.

It is interesting to adventurously click around the world map. There's some illuminating content out there you might never have found otherwise.
